# GuideTranslator Enterprise Sales Tool

Interactive sales calculator for cruise line executives. Calculates cost savings from AI-powered real-time translation vs. traditional multilingual guides.

## Features
- Landing page with value proposition
- Lead capture & registration
- Interactive fleet calculator with 9 parameters
- Persistent calculation storage (Supabase)
- Contact/offer request flow

## Tech Stack
- React 18 + Vite
- Supabase (lead persistence)
- Deployed on Vercel

## Setup
```bash
npm install
cp .env.example .env
# Add your Supabase anon key
npm run dev
```

## Domains
- `app.guidetranslator.com` → Sales tool (this app)
- `guidetranslator.com` → Landing page
